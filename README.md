# Análisis de Rendimiento de Tiendas - Alura Latam
# Descripción del Proyecto
Este proyecto es un análisis exploratorio de datos (EDA) desarrollado en Python a través de un Jupyter Notebook. El objetivo principal es evaluar y comparar el rendimiento de cuatro sucursales o tiendas distintas utilizando bases de datos proporcionadas en el Challenge 1 de Data Science Latam de Alura.

# Tecnologías Utilizadas
Para ejecutar este análisis, se hizo uso de las siguientes herramientas y librerías de Python:

- Python 3

- Pandas: Para la importación, manipulación y cálculo de las métricas sobre los conjuntos de datos.

- Matplotlib (pyplot): Para la visualización de los datos mediante gráficos de barras.

# Estructura del Análisis
El código está dividido en cinco secciones principales de análisis de negocio:

- Análisis de facturación: Calcula y grafica los ingresos totales (en COP) generados por cada una de las cuatro tiendas.

- Ventas por categoría: Cuenta la cantidad de productos vendidos agrupados por categoría para cada tienda, representados en un panel de gráficos.

- Calificación promedio de la tienda: Evalúa el nivel de satisfacción de los clientes calculando la media de las calificaciones (en una escala del 1 al 5) para cada sucursal.

- Productos más y menos vendidos: Identifica e ilustra el "Top 5" de productos con mayor volumen de ventas y los 5 con menor volumen por cada tienda.

- Envío promedio por tienda: Analiza y compara el costo promedio de envío que asume cada sucursal.

# Fuente de Datos
Los datos son extraídos directamente de los repositorios de GitHub de Alura en formato .csv. Se analizan cuatro archivos independientes, correspondientes a:

- tienda_1.csv

- tienda_2.csv

- tienda_3.csv

- tienda_4.csv

# Cómo usar este proyecto
Asegúrate de tener instalado Python en tu entorno junto con las librerías pandas y matplotlib.

1. Descarga o clona este repositorio.

2. Abre el archivo AluraStoreLatamJean.ipynb utilizando Jupyter Notebook, Jupyter Lab o Google Colab.

3. Ejecuta las celdas en orden. El código descargará los datos automáticamente desde internet (no requieres los archivos CSV localmente) y generará los gráficos y resultados en consola.
